A CLI tool that gives LLMs REPL-like access to a live web page through real Cypress commands

65+ commands • Aria snapshots • Cypress test export • AI-agent ready

$ npm install -g cypress-cli
Try the Demo Read the Docs

"Every action executes as a genuine Cypress command, and the result is returned as an aria snapshot — a compact, accessibility-tree-based view of the page that LLMs can reason about."

What makes it different

Real Cypress Commands

cy.click(), cy.type(), cy.visit() — not simulated. Every action executes as a genuine Cypress command in a live browser.

Aria Snapshots

Accessibility-tree DOM representation. Compact, semantic, and LLM-readable. Ported from Playwright's aria snapshot implementation.

Test Generation

Export your entire session as a ready-to-run .cy.ts test file. Every command maps 1-to-1 to a Cypress API call.

Three-column workspace

  • Commands
  • Assertions
  • Network
  • Storage
  • Navigation 
$ cypress-cli open https://example.com
$ cypress-cli snapshot
$ cypress-cli click e5
$ cypress-cli type e3 "hello world"
$ cypress-cli assert e5 contain.text "hello"
$ cypress-cli export --file my-test.cy.ts

Tool-use flow

→ cypress-cli open https://app.example.com
← Page URL: https://app.example.com
← Page Title: Dashboard — My App
← [Snapshot](.cypress-cli/page-...679Z.yml)

→ cypress-cli click e2
← Page URL: https://app.example.com/settings
← # Ran Cypress code:
← cy.get('[data-cy="settings"]').click()
← [Snapshot](.cypress-cli/page-...123Z.yml)

→ cypress-cli export --file settings.cy.ts
← Wrote test file: settings.cy.ts
